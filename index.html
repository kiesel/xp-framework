    • XP Framework

    The XP Framework is an all-purpose, object-oriented PHP framework.

    Project maintained by kiesel Hosted on GitHub Pages — Theme by mattgraham

    About

    The XP framework offers consistent, multi-purpose, object oriented, production-tested collection of classes, APIs for app server connectivity, web services, dynamic web sites, date access and manipulation, logging, collections, I/O, databases, O/R mapping, XML, unittesting, and much more.

    Example: Age in days

    <?php
  uses('util.Date', 'util.DateUtil');

  class AgeInDays extends Object {
    public static function main(array $args) {
      $span= DateUtil::timespanBetween(new Date($args[0]), Date::now());
      Console::writeLine('Hey, you are ', $span->getDays(), ' days old');
    }
  }
?>

    Now, run it:

    $ xp AgeInDays 1980-05-28
Hey, you are 11699 days old
